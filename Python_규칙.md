# Python 기본 규칙 (pep-8)



- 들여쓸 때는 4칸 들여쓰기 
- : 앞에는 분이고 뒤에는 띄어쓰기
- 리스트, 딕셔너리 마지막 값 뒤에도 쉼표
  - 기본적으로 한 줄에 값 하나씩 입력
- 되도록이면 `'  '` 작은 따옴표 사용





- 변수/ 함수 이름은 snake_case

  `snake_case = '변수'`

  `def function_snake_case():`



- class 이름은 CamelCase

  `class CamelCase:`



- 키워드 인자로 사용할 때는 붙여쓰고  / 변수에 할당할 때는 띄어쓰기

  `dict(a='apple', b='banana')` 			   ⭕

  `dict( a = 'apple' , b = 'banana' )` ❌

  

  `a = 3 + 5` 	⭕

  `a=3+5` 			❌
